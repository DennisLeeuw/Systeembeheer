Een grafische interface met alleen een pointer heeft niet veel nut. We willen graag de mogelijkheid hebben om applicaties op te starten, een klok om de tijd te zien en een mogelijkeid om bestanden te benaderen. Deze totale omgeving wordt de desktop\index{desktop} of het buroblad\index{buroblad} genoemd.

De desktop is wat de meeste mensen kennen als de GUI. De desktop is meer dan alleen een grafische interface. De desktop is dus een collectie applicaties netzo als de CLI een collectie commando's is.
